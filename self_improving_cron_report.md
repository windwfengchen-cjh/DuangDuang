## 🔧 配置详情

### Cron表达式解析
```
10 18 * * *
│  │  │ │ │
│  │  │ │ └── 星期 (0-7, 每天)
│  │  │ └──── 月份 (1-12, 每月)
│  │  └────── 日期 (1-31, 每日)
│  └───────── 小时 (18, 18点)
└──────────── 分钟 (10, 10分)
```
**含义**: 每天的 18:10 执行一次
