# Find Skills 技能

## 技能信息
- **技能ID**: find_skills
- **版本**: 1.0.0
- **创建日期**: 2026-03-19

## 功能描述
查找和发现可用的技能，支持本地和远程搜索。该技能使智能体能够：
1. 扫描本地技能仓库中的所有可用技能
2. 查询远程技能仓库
3. 根据关键词搜索匹配的技能
4. 显示技能的详细信息和状态
5. 支持技能版本检查和兼容性验证

## 执行流程

### 1. 本地技能扫描
- 读取 `.clawhub/lock.json` 获取已安装技能列表
- 遍历 `skills/` 目录下的所有技能文件夹
- 读取每个技能的 `_meta.json` 获取元数据
- 验证技能完整性（检查 SKILL.md 是否存在）

### 2. 远程技能仓库查询
- 连接远程技能仓库 API
- 获取可用技能列表
- 检查新版本和更新

### 3. 技能搜索
- 按名称搜索：匹配技能ID和名称
- 按标签搜索：匹配技能标签
- 按分类搜索：按类别筛选
- 模糊搜索：支持部分匹配

### 4. 结果显示
- 表格形式展示技能列表
- 显示版本、状态、安装时间
- 标记可更新技能

### 5. 详细信息获取
- 读取 SKILL.md 内容
- 显示功能描述和执行流程
- 显示依赖和兼容性信息

## 使用方法

### 列出所有本地技能
```bash
# 通过子智能体执行
子智能体: 执行 find_skills 技能，列出所有本地技能
```

### 搜索特定技能
```bash
子智能体: 执行 find-skills 技能，搜索关键词 "forward"
```

### 检查技能更新
```bash
子智能体: 执行 find-skills 技能，检查远程更新
```

## 文件结构
```
skills/find_skills/
├── SKILL.md              # 本文件
├── _meta.json            # 技能元数据
└── .clawhub/
    └── origin.json       # 安装来源信息
```

## 触发方式
- **手动触发**: 通过子智能体调用
- **命令行**: 使用 skill_find.py 脚本

## 依赖
- Python 3.8+
- 访问 workspace/skills/ 目录权限
- 访问 .clawhub/lock.json 权限
- 可选：远程仓库 API 访问
