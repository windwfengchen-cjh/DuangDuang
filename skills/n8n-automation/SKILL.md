# 自动化工作流 (n8n-automation)

## 概述

设计n8n工作流JSON，含触发器/重试/错误处理/人工审核。

## 功能

- n8n工作流设计
- 多种触发器支持
- 错误重试机制
- 人工审核节点
- 工作流导入导出

## 工作流组件

### 触发器
- Webhook
- 定时调度
- 事件监听
- 手动触发

### 节点类型
- HTTP请求
- 数据处理
- 条件分支
- 循环执行
- 错误处理
- 人工审核

### 集成服务
- 飞书
- Slack
- GitHub
- 数据库
- API接口

## 使用方法

1. 定义业务场景
2. 设计工作流节点
3. 配置参数和连接
4. 生成JSON配置
5. 导入n8n执行

## 输出格式

```json
{
  "name": "工作流名称",
  "nodes": [...],
  "connections": {...},
  "settings": {...}
}
```

- n8n平台
- API集成能力
