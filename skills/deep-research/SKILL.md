# 深度研究 (deep-research)

## 概述

多引擎搜索+网页提取+结构化分析报告。

## 功能

- 多搜索引擎并行查询
- 网页内容智能提取
- 结构化分析报告生成
- 研究主题深度挖掘

## 使用方法

1. 指定研究主题和关键词
2. 自动执行多引擎搜索
3. 提取并分析相关内容
4. 生成结构化研究报告

## 输入参数

- `topic`: 研究主题
- `keywords`: 关键词列表
- `depth`: 研究深度 (basic/standard/deep)
- `sources`: 来源数量限制

## 输出格式

```markdown
# 研究报告: [主题]

## 摘要
...

## 主要发现
...

## 详细分析
...

## 结论与建议
...

## 参考来源
...
```

- 内容提取工具
- 文档生成工具
