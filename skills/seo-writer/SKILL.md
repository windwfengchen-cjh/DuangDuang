# SEO内容写作 (seo-writer)

## 概述

SEO优化的长文写作，关键词策略+结构化内容。

## 功能

- 关键词研究与分析
- SEO优化内容创作
- 标题与元描述优化
- 内容结构优化

## 使用方法

1. 输入目标关键词
2. 分析关键词竞争度
3. 生成SEO优化大纲
4. 撰写并优化内容

## SEO优化要素

- 标题包含主关键词
- 合理的H标签结构
- 关键词密度控制(1-3%)
- 内部链接布局
- 元描述优化
- 图片ALT标签

## 内容结构模板

```
H1: 主标题 (含关键词)
  └─ 引言
  └─ H2: 主要章节1
      └─ H3: 子章节
  └─ H2: 主要章节2
  └─ H2: 总结
  └─ 相关链接
```
