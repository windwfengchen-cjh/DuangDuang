# 飞书互动卡片 (feishu-cards)

## 概述

发送按钮、表单、投票、链接卡片。

## 功能

- 多种卡片模板
- 交互式按钮
- 表单收集
- 投票功能
- 链接预览

## 卡片类型

### 1. 文本卡片
- 标题+内容+按钮

### 2. 表单卡片
- 输入框
- 选择器
- 日期选择
- 提交按钮

### 3. 投票卡片
- 单选/多选
- 实时统计
- 匿名/实名

### 4. 链接卡片
- 网页预览
- 图片+标题+描述

## 使用方法

```python
# 发送文本卡片
card = {
    "header": {"title": "标题"},
    "elements": [
        {"tag": "div", "text": "内容"},
        {"tag": "action", "actions": [{"tag": "button", "text": "点击"}]}
    ]
}
```

## 输入参数

- `card_type`: 卡片类型
- `title`: 标题
- `content`: 内容
- `buttons`: 按钮配置
- `target`: 发送目标
