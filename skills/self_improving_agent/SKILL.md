# Self-Improving Agent Skill

A comprehensive self-analysis and continuous improvement system for AI agents.

## Version

**2.0.0** - Major upgrade with enhanced analysis capabilities and actionable improvement generation.

## Description

This skill enables AI agents to perform deep self-analysis, identify improvement opportunities, and generate actionable recommendations for enhancing performance, capabilities, and effectiveness.

## When to Use

- After completing significant tasks or projects
- During periodic self-assessments
- When encountering recurring challenges or errors
- To identify skill gaps and learning opportunities
- For continuous capability enhancement

## Features

### Core Capabilities
- **Performance Analysis**: Review execution history and identify patterns
- **Error Pattern Detection**: Analyze mistakes and their root causes
- **Skill Gap Assessment**: Identify missing capabilities and knowledge
- **Improvement Recommendation**: Generate specific, actionable suggestions
- **Progress Tracking**: Monitor improvement over time

### Analysis Dimensions
1. **Execution Efficiency**: Speed, accuracy, resource usage
2. **Decision Quality**: Judgment calls, trade-offs, outcomes
3. **Communication**: Clarity, tone, appropriateness
4. **Tool Usage**: Effectiveness of tool selection and application
5. **Learning Velocity**: How quickly new skills are acquired

## Usage

### Basic Self-Analysis

```
Run self-improving agent analysis
```

### Targeted Analysis

```
Analyze my recent performance on [specific task type]
```

### Deep Dive

```
Perform comprehensive self-improvement review
```

## Output Format

The skill generates a structured report including:

1. **Executive Summary**: High-level findings and priorities
2. **Strengths**: What's working well
3. **Areas for Improvement**: Specific weaknesses identified
4. **Root Cause Analysis**: Why issues occur
5. **Action Items**: Concrete steps for improvement
6. **Learning Recommendations**: Skills to acquire or enhance

## Implementation

The skill operates by:
1. Reading relevant memory files and execution logs
2. Analyzing patterns in past interactions
3. Comparing against best practices and optimal patterns
4. Generating prioritized improvement recommendations
5. Documenting findings for future reference

## Integration

Works seamlessly with:
- Memory system for historical data access
- Skill management for capability assessment
- Task tracking for performance metrics
- Configuration files for behavior analysis

## Best Practices

1. **Regular Reviews**: Run weekly or bi-weekly for continuous improvement
2. **Specific Focus**: Target specific areas rather than broad analysis
3. **Action-Oriented**: Prioritize recommendations with clear next steps
4. **Track Progress**: Compare current analysis with previous reviews
5. **Document Insights**: Save key learnings to MEMORY.md

## Requirements

- Access to memory files (memory/*.md, MEMORY.md)
- Access to skill configurations (skills/*/SKILL.md)
- Access to workspace files (SOUL.md, AGENTS.md, USER.md)
- Write access for generating reports
