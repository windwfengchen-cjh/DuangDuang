# 今日热榜 (trending)

## 概述

微博/知乎/抖音/百度等各平台热搜榜单，一键获取全网热点。

## 功能

- 多平台热搜聚合
- 实时热点监控
- 历史趋势分析
- 热点分类筛选

## 支持平台

- 微博热搜
- 知乎热榜
- 抖音热榜
- 百度热搜
- 今日头条
- B站热门
- 快手热榜
- 微信指数

## 使用方法

1. 指定平台或获取全部
2. 可选关键词筛选
3. 获取热榜数据

## 输出格式

```json
{
  "platform": "微博",
  "timestamp": "2026-03-19T19:20:00+08:00",
  "trends": [
    {
      "rank": 1,
      "title": "热点标题",
      "heat": 1000000,
      "url": "https://..."
    }
  ]
}
```

## 输入参数

- `platform`: 平台名称 (weibo/zhihu/douyin/baidu/all)
- `limit`: 返回条数
- `category`: 分类筛选
