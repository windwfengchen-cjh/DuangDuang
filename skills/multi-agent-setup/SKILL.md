# 多Agent团队配置 (multi-agent-setup)

## 概述

一只龙虾变一支团队——配置多个AI角色协作，总指挥+笔杆子+参谋+运营官。

## 功能

- 配置多个AI Agent角色协作
- 总指挥：统筹协调，任务分配
- 笔杆子：内容创作，文案撰写
- 参谋：分析建议，决策支持
- 运营官：执行落地，日常运营

## 使用方法

1. 在 AGENTS.md 中定义各Agent角色和职责
2. 使用 coordinator 角色分配任务
3. 通过子Agent执行具体工作
4. 汇总结果并整合输出

## 配置示例

```markdown
## 总指挥 (Coordinator)
- 职责：任务拆解、资源协调、质量把控
- 工具：所有工具

## 笔杆子 (Writer)
- 职责：内容创作、文案优化
- 工具：文档、搜索、写作工具

## 参谋 (Advisor)
- 职责：数据分析、策略建议
- 工具：分析、研究工具

## 运营官 (Operator)
- 职责：执行落地、日常维护
- 工具：自动化、监控工具
```

## 依赖

- AGENTS.md 配置
- 子Agent支持
