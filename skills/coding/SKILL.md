# 代码助手 (coding)

## 概述

Claude Code/Codex写代码、改Bug、部署。PTY模式自动应答。

## 功能

- 代码编写与重构
- Bug诊断与修复
- 代码审查
- 自动化部署
- 技术方案设计

## 支持的编程语言

- Python
- JavaScript/TypeScript
- Java
- Go
- Rust
- C/C++
- SQL
- Shell
- 其他主流语言

## 使用方法

### 写代码
```
描述需求 -> 生成代码 -> 解释逻辑 -> 测试验证
```

### 改Bug
```
描述问题 -> 分析原因 -> 提供修复 -> 验证解决
```

### 代码审查
```
提交代码 -> 审查问题 -> 给出建议 -> 优化改进
```

## PTY模式

- 自动应答交互式命令
- 支持长时间运行任务
- 实时输出反馈

## 输入参数

- `task`: 任务类型 (write/fix/review/deploy)
- `language`: 编程语言
- `description`: 需求描述
- `code`: 现有代码

## 输出

- 代码文件
- 修改说明
- 使用文档
- 测试用例
