# 工作汇报生成 (summary-report)

## 概述

从Session历史自动生成工作总结PDF，支持日报/周报。

## 功能

- 自动分析工作历史
- 生成结构化汇报
- 支持多种格式(PDF/Markdown)
- 日报/周报模板

## 汇报类型

### 日报
- 今日完成工作
- 进行中任务
- 遇到的问题
- 明日计划

### 周报
- 本周概览
- 重点项目进展
- 数据指标
- 下周计划
- 需要支持

## 使用方法

1. 指定汇报类型和时间段
2. 系统自动分析历史记录
3. 生成汇报草稿
4. 人工确认后发送

## 输出格式

```markdown
# 工作日报 - 2026年3月19日

## 今日完成
- [x] 任务1
- [x] 任务2

## 进行中
- [ ] 任务3

## 问题与风险
...

## 明日计划
...
```

- PDF生成工具
