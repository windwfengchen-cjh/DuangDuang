# 小红书运营 (xiaohongshu)

## 概述

搜索笔记、发布内容、分析评论。基于xiaohongshu-mcp(8400+⭐)。

## 功能

- 小红书笔记搜索
- 内容发布
- 评论分析
- 热点追踪
- 账号运营

## 使用方法

### 搜索笔记
```
搜索关键词 -> 获取笔记列表 -> 分析内容
```

### 发布内容
```
创建内容 -> 选择话题 -> 发布笔记
```

### 数据分析
```
获取笔记数据 -> 分析互动指标 -> 优化策略
```

## 内容创作指南

### 标题技巧
- 使用数字
- 情感词汇
- 悬念设置
- 痛点切入

### 内容结构
- 开头抓眼球
- 中间有价值
- 结尾引导互动

### 标签策略
- 热门标签
- 精准标签
- 品牌标签

## 输入参数

- `action`: 操作类型 (search/publish/analyze)
- `keyword`: 搜索关键词
- `content`: 发布内容
- `tags`: 标签列表
