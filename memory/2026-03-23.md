### Auto-forward Duplicate Detection (Implemented)
- **Feature**: `auto_forward.py` now has `find_similar_record()` and `update_record_with_supplement()`
- **Logic**: 
  - Extract keywords (order IDs, product names, locations)
  - Search last 7 days for similar records (similarity threshold 0.6)
  - If match found → update existing record
  - If no match → create new record
- **Similarity factors**:
  - Order ID match: 100% similarity
  - Product keywords: 50% weight
  - Location keywords: 30% weight
  - Text overlap: 20% weight
